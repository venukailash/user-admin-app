## File Descriptions

### `.gitignore`
- **Purpose:** Specifies intentionally untracked files that Git should ignore.

### `.nvmrc`
- **Purpose:** Contains the version of Node.js recommended for use with the project.

### `.prettierrc`
- **Purpose:** Configuration file for Prettier, a code formatter.

### `eslint.config.js`
- **Purpose:** Configuration file for ESLint, a static code analysis tool for identifying problematic patterns found in ECMAScript/JavaScript code.

### `index.html`
- **Purpose:** The main HTML file of the project.

### `package-lock.json` and `package.json`
- **Purpose:** Contains information about the project's dependencies and scripts.

### `README.md`
- **Purpose:** A markdown file containing documentation or instructions for the project.

### `tsconfig.app.json`, `tsconfig.json`, `tsconfig.node.json`
- **Purpose:** TypeScript configuration files for different environments (app, root, node).

### `vite.config.ts`
- **Purpose:** Configuration file for Vite, a build tool for modern web development.

### `coverage/`
- **Purpose:** Directory containing code coverage reports.

### `public/`
- **Purpose:** Public assets of the application.

### `src/App.css`
- **Purpose:** Styles for the main App component.

### `src/App.test.tsx`
- **Purpose:** Tests for the App component using Vitest and React Testing Library.

### `src/App.tsx`
- **Purpose:** The main application component that includes a navbar, routes, and footer.

### `src/assets/`
- **Purpose:** Directory for project assets like images or fonts.

### `src/components/`
- **Purpose:** Contains reusable UI components such as `FailureMessage`, `Footer`, and `Navbar`.

### `src/hooks/`
- **Purpose:** Contains custom hooks for use in the application.

### `src/pages/`
- **Purpose:** Contains different pages of the application like `Login` and `Details`.

### `src/services/`
- **Purpose:** Contains services or utility functions used throughout the application.

### `src/tests-setup.mts`
- **Purpose:** Setup file for testing with Vitest.
